# Heap Data Structures Assignment Report

## Heapsort Analysis

Time Complexity:
- Best Case: O(n log n)
- Average Case: O(n log n)
- Worst Case: O(n log n)

Space Complexity:
- O(1) auxiliary space

## Priority Queue

Operations:
- insert: O(log n)
- extract_max: O(log n)
- increase_priority: O(log n)
- is_empty: O(1)

## Applications

- CPU Scheduling
- Event Simulation
- Network Routing
